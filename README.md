# Real Estate Website

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Features

- View available properties
- Search and filter properties by location, price, type, bedrooms and bathrooms
- View detailed property information
- View multiple property images in a gallery
- Register and log in as a user
- Save and remove favourite properties
- Submit property enquiries
- Submit general contact enquiries
- Admin dashboard
- Add new properties
- Upload multiple property images
- Edit existing properties
- Delete properties
- View customer enquiries
- Responsive layout for desktop, tablet and mobile devices

## Local Setup

1. Install XAMPP.
2. Place the project folder inside:

C:\xampp\htdocs
